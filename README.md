# Cotizador de Medicamentos y Despacho

Este es un script interactivo en Python desarrollado para calcular el costo total de medicamentos y despacho, aplicando descuentos específicos basados en la edad y el tramo de salud del usuario (A, B, C o D).
Diseñado en base a lo solicitado en la evaluación.

## Características
- Validación de entradas (edad como número entero mayor de 18 años).
- Control de tramos válidos mediante bucles dinámicos.
- Aplicación acumulativa de descuentos por edad y tramos de salud.

## Requisitos
- Python 3.x
