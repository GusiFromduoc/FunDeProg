# Cotizador de Medicamentos y Despacho / Juego de adivina el número.

Estos dos son script interactivo en Python desarrollado para calcular el costo total de medicamentos y despacho, aplicando descuentos específicos basados en la edad y el tramo de salud del usuario (A, B, C o D).
Mientras que el otro es un juego de adivinar un número entre un rango establecido por el usuario.
Ambos diseñados en base a lo solicitado en la evaluación.

## Características
- Validación de entradas (edad como número entero mayor de 18 años).
- Control de tramos válidos mediante bucles dinámicos.
- Aplicación acumulativa de descuentos por edad y tramos de salud.
--------------------------------------------------------------------
- Validación de entradas (un número inferior y superior con lógica a su nombre).
- Control de intentos validos mediante bucles dinámicos para que los intentos siempre tengan lógica.
- Uso de el uso de funciones importadas desde random.

## Requisitos
- Python 3.x
